:: EXAMS ::
Contains info about patients exams
mpi:: unique identifier for the patient
ulss:: health structure a patient belongs to
req_number:: request number
sampling_requestor:: who require the sampling
microbiology_code:: laboratory code
microbiology_name:: laboratory name
sampling_date:: date of the sampling
sampling_response_date:: date of the sampling response
result:: result "POSITIVO" means the sampling is positive, "NEGATIVO" the sampling is negative, DOUBT otherwise
method:: sometimes there is a specific methology associated with the sampling
